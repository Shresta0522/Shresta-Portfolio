      • Screen-1: Course Page
      • The Page is divide into two coloums where the first column includes the course form and the second column includes the list of courses created.
      • In the first column, user will be able to add information in the course form which take the course details of course description,course name, coursetype whether IT or SS,semester course start date and end date and course instructors.
      • The course decription includes the markdown preview,which helps the user to preview the data based on the text highlight like bold,italic,ordered unorderd etc.
      • The second column must display all the course name which are already created. ON clicking the course name button it must display the open,edit and delete button options.
      Screen-2: Learning Objectives Page
      • The Page is divide into two coloums where the first column includes the Learning objective form and the second column includes the list of learning objective buttons created.
      • Learning objective form includes the unique id along with the course name, descritpion, activities to be added like video url,assignment,quiz and notes. on clicking add button the activites should be created.
      • Activites form include the activity id and the type of activity. If the activity is a video it includes the video url and title. If the activity is either notes or assignemnt it includes the description.
      • On clicking the create button a JSON file must be created with a learning objective id as the file name.
      Screen-3: Day Module Page
      • The Page is divide into two coloums where the first column includes the Day Module form and the second column includes the list of Day modules created.
      • Day module form includes the unique id along with the course name, descritpion. Along with 2 dropdown buttons which is to add the learning objecties and its pre-requites for that particular learning objective.
      • All the learning objective and its pre-requites will be add as a one row in the table.
      • On clicking the create day module,it creates a JSON file and will display on the right column in the list of day modules created.
