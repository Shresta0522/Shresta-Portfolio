User Story
    As a course developer, I should be able to link the pre-requisite learning objectives for a learning objective, so that it can be used to personalize the module for each student in the iLMS.

Day Modules page

Description:
    Day-Wise Module
    • n daywise modules, we have created a form which consists of id and description and given a two select dropdowns where one is for learning objective and other is for prerequisites
    • In the select dropdown, the learning objectives were fetched from backend using api calls. After selecting the learning objectives then we need to click on add and to preview that we have displayed it in table below. After entering all the details we need to click on save button
    • Once we click on Save button,the json get created and gets appended to the file. These files will be stored in one folder which is created based on name of the course.
    List of Day Modules
    • On clicking create button, it is previewed at the right of the page and clicking on it, it displays the items like preview and delete. When we click on preview then the details in the file gets displayed. When we click on delete then that files get deleted.
    • When user clicks on course page then the page gets redirected to course page and when user clicks on module page it redirects to the day module page.
    • In day modules page we created a button and when user clicks on it then the page redirects to learning objectives page. Therefore, this is how we were able to connect course page, Learning objective page and the day module page.
