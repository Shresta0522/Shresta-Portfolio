Learning Objectives Page

Description:
    Generating ID
    • Worked on creating Learning objective which has series attached with the related course name. For example: Python-L1,Python-L2, Etc. L represents the learning objective and followed by the number for it. Using Api we were send the course name from one component to another as the course database will have all the information
    • Also worked on generating Youtube title baed on the Youtube ID provided in the input feild. Where the yoube links are one of the activities in learning objectives
    Create,Save and Display JSON Content
    • Here,we have used multer in node to save the file in JSON format and used file storage in node.js to save it in particular path.
    • In the backend, it will search for the folder as per course name and if folder doesn't exist then it will create the folder and then creates the Learning Objective Files as JSON
    • It creates a folder and then creates the files later in it. the JSON data gets appended to to that folder. Along with this, the dropdown will be displayed at the right of the page.
    • In learning objective page, when the user clicks on the dropdown button then it displays the items like preview and delete When user clicks on preview then it displays the id, title, activity type, video id, notes title and when the user clicks on delete then that file gets deleted.
    • To display all the learning objective files we have used apis call from the backend to display all the files.
